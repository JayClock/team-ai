===== SSE Response Format

The response is streamed as Server-Sent Events with the following event types:

|===
|Event Type |Description

|`start`
|Indicates the beginning of the response stream

|`text-start`
|Indicates the beginning of text content

|`text-delta`
|Contains a chunk of text content in the `delta` field

|`text-end`
|Indicates the end of text content

|`finish`
|Indicates the end of the response stream

|`[DONE]`
|Final marker indicating stream completion
|===
